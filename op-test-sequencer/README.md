# op-test-sequencer

This is a test service for block sequencing.
This service is in active development.

## Usage

### Build from source

```bash
# from op-test-sequencer dir:
just op-test-sequencer
./bin/op-test-sequencer --help
```

### Run from source

```bash
# from op-test-sequencer dir:
go run ./cmd --help
```

### Build docker image

Not available yet.
